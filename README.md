# KwikMarket

A real-time market coordination platform digitizing traditional African (Ghana-focused) markets.

## Features

- **Consumer App**: Pre-order items, track orders, inspect and pay
- **Vendor Portal**: Manage catalog, accept orders, mark ready
- **Shopper Module**: Accept jobs, pickup and deliver orders
- **Admin Panel**: User verification, dispute resolution, analytics

## Getting Started

```bash
npm install
npm run dev
```

## Security Notice

> **⚠️ Important:** Before deploying to production, enable **leaked-password protection** in your Supabase project settings.
>
> This feature prevents users from using passwords that have been exposed in known data breaches.
>
> Learn more: [Supabase Auth Security](https://supabase.com/docs/guides/auth/password-security)

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth, Database, Edge Functions, Storage)
- Framer Motion for animations
